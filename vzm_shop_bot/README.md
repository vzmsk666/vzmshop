# VZM SHOP Telegram Bot (MVP)

## Что делает бот
- Показывает каталог услуг (Буст ранга, Коучинг, Разбор реплея, Игра с VZM)
- Оформляет заявки без сбора данных (всё через оператора)
- Для буста ранга: выбирается режим (2v2 активен), тип буста, стартовый/целевой ранг+дивизион, бот считает цену
- Создаёт заказ и отправляет карточку заказа в админ-чат
- В админ-чате есть кнопки статусов: Новый / Связались / Оплата / В работе / Готово / Отмена
- Хранит заказы в SQLite

## Быстрый запуск локально
1) Установите Python 3.11+
2) В папке проекта:
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

3) Создайте файл `.env` (см. `.env.example`) и заполните:
- BOT_TOKEN
- ADMIN_CHAT_ID
- SUPPORT_USERNAME (опционально)

4) Запуск:
```bash
python -m bot
```

## Деплой
Подойдёт любой хостинг, который поддерживает long polling (Render/Fly/Railway/VPS).
Webhook можно добавить позже (архитектура совместима).

## Важно
- Бот НЕ принимает оплату, НЕ принимает файлы, НЕ собирает ник/discord и т.д.
- Всё делает оператор после заявки.
